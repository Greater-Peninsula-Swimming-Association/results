# CLAUDE.md

Guidance for Claude Code when working with this repository.

---

## Repository Overview

GPSA meet results archive, served at `results.gpsaswimming.org`. Contains all dual meet result HTML files organized by season year, historical invitational meet files, and the scripts that build and maintain the archive indexes.

---

## Structure

```
/
├── 2022/                    # Dual meet results by season year
├── 2023/
├── 2024/
├── 2025/
│   ├── YYYY-MM-DD_T1_v_T2.html   # Individual meet result files
│   ├── divisions.csv              # Division assignments for archive builder
│   └── index.html                 # Auto-generated season archive (do not edit)
├── 2026/
├── invitationals/           # Historical invitational static files (PDFs, meet invites)
│   ├── CityMeet/
│   ├── SummerSplashInvitational/
│   └── MiniMeet/
├── index.html               # Root index — auto-generated by generate-indexes workflow
└── scripts/
    ├── build_archive.py     # Season archive generator
    ├── generate_index.py    # Directory index page generator
    └── bulk_process_results.py
```

---

## File Naming

Dual meet results must follow: `YYYY-MM-DD_TEAM1_v_TEAM2.html`

---

## GitHub Actions

### build-season-archive.yml
Triggers when result HTMLs or `divisions.csv` change in a year directory. Runs `build_archive.py` and commits the updated `index.html` for that year. Can be manually dispatched for a specific year.

### generate-indexes.yml
Triggers on any push (excluding index.html, scripts, and config changes). Runs `generate_index.py` from the repo root to regenerate directory indexes for root and all `invitationals/` subdirectories. Commits with `[skip ci]` to avoid loops.

---

## Adding Meet Results

1. Add the result HTML file to the correct year directory: `YYYY/YYYY-MM-DD_T1_v_T2.html`
2. Ensure `divisions.csv` exists in that year directory
3. Push — `build-season-archive.yml` auto-regenerates the season archive

## Adding Invitational Files

1. Add files to the appropriate `invitationals/` subdirectory
2. Push — `generate-indexes.yml` auto-updates the directory index

---

## Shared CSS

All generated pages reference `https://css.gpsaswimming.org/gpsa-tools-common.css`. Do not add local CSS files.

---

## Scripts

See `scripts/` for documentation within each file. Run locally with:

```bash
# Rebuild a season archive manually
python scripts/build_archive.py -i 2025 -o 2025 --non-interactive

# Regenerate all directory indexes
python scripts/generate_index.py .
```
